# RepoSpaceShooter

## How to Play!
**Movement** <br>
W / Up arrow key: Drive forward <br>
A / Left arrow key: Turn Left   <br>
S / Down arrow key: Reverse     <br>
D / Right arrow key: Turn Right <br>

**Shooting**  <br>
Space: Shoot  <br>

*Have fun shooting as many asteroids as possible! And don't crash into them!*
